## আইকন তৈরির উপায়:

### অনলাইন টুল (সহজ):
1. https://www.pwabuilder.com/imageGenerator এ যান
2. একটি ছবি আপলোড করুন (512x512 বা বড়)
3. সব সাইজ ডাউনলোড করুন
4. `public/icons/` ফোল্ডারে রাখুন

### ম্যানুয়াল (Photoshop/GIMP):
1. একটি 512x512 ছবি তৈরি করুন
2. উপরের সব সাইজে রিসাইজ করুন
3. PNG ফরম্যাটে সেভ করুন
4. `public/icons/` ফোল্ডারে রাখুন

## ডিফল্ট আইকন:
যদি এখনই আইকন না বানান, তাহলে ব্রাউজার ডিফল্ট আইকন দেখাবে।
পরে যেকোনো সময় আইকন যোগ করতে পারবেন।
