# 🚀 দ্রুত শুরু করুন (Quick Start)

## ১. API URL সেট করুন

`services/api.js` ফাইলে যান এবং এই লাইন পরিবর্তন করুন:

```javascript
const API_URL = 'http://192.168.1.100:3000';
```

### আপনার IP বের করুন:
1. Windows Start মেনু খুলুন
2. `cmd` টাইপ করুন
3. `ipconfig` কমান্ড চালান
4. `IPv4 Address` খুঁজুন (যেমন: 192.168.1.100)

---

## ২. অ্যাপ চালান

```bash
cd native-mobile-app
npm start
```

একটি QR code দেখাবে।

---

## ৩. মোবাইলে ওপেন করুন

### Android:
1. Play Store থেকে **Expo Go** ডাউনলোড করুন
2. Expo Go অ্যাপে QR code স্ক্যান করুন

### iPhone:
1. App Store থেকে **Expo Go** ডাউনলোড করুন
2. Camera দিয়ে QR code স্ক্যান করুন
